Como funciona

Protocolo x402 — pagamento nativo em HTTP, sem intermediários.

1

Request

Agente faz GET /v1/bcb/selic

2

402 Payment Required

API retorna preço e endereço de pagamento em USDC (Base network)

3

Pagamento on-chain

SDK paga automaticamente e envia header X-PAYMENT

4

200 OK + Dados

Dados retornados em JSON — prontos para consumo

Quick Start

Integre em minutos — o SDK x402 cuida do pagamento automaticamente.

cURL
Python
TypeScript
# 1. Primeira chamada retorna 402 com dados de pagamento
curl -s https://databr.api.br/v1/bcb/selic
# {"x402Version":1,"accepts":[{"scheme":"exact","network":"eip155:8453",
#   "maxAmountRequired":"3000","asset":"0x833589...","payTo":"0x..."}]}

# 2. Use o x402 SDK para pagamento automático:
#    npm install x402 / pip install x402 / go get github.com/coinbase/x402/go

# 3. Com o SDK, a chamada é transparente:
#    client.get("https://databr.api.br/v1/bcb/selic")
#    → Paga automaticamente → Retorna dados JSON
from x402.client import PaymentClient

# Configure com sua wallet (USDC na rede Base)
client = PaymentClient(
    private_key="0xYOUR_PRIVATE_KEY",
    network="base-mainnet"
)

# O SDK cuida do fluxo 402 → pagamento → retry automaticamente
resp = client.get("https://databr.api.br/v1/bcb/selic")
data = resp.json()

print(data["data"]["valor"])   # 14.25
print(data["cost_usdc"])        # "0.003"
import { createx402Client } from "x402";

// Configure com sua wallet (USDC na rede Base)
const client = createx402Client({
  privateKey: process.env.WALLET_PRIVATE_KEY,
  network: "base-mainnet",
});

// Pagamento automático via x402
const resp = await client.get(
  "https://databr.api.br/v1/empresas/33000167000101"
);

console.log(resp.data.razao_social); // "PETROLEO BRASILEIRO S.A."
console.log(resp.cost_usdc);         // "0.003"
